HTML 5 : Le squelette des sites web

Grafikart 17 vidéos

Apprendre le HTML

Le HTML (HyperText Markup Language) est le langage de balisage standard pour créer des pages web. Il définit la structure et le contenu d'une page à l'aide d'éléments comme les titres, paragraphes, liens, images et listes. Cette formation complète couvre tout, des balises de base aux éléments sémantiques HTML5, vous enseignant comment créer du contenu web bien structuré et accessible. Vous apprendrez les bonnes pratiques pour organiser votre balisage, comprendre le modèle DOM, et préparer une base solide pour le styliser avec CSS et ajouter de l'interactivité avec JavaScript.

CSS : Essentiel pour la personnalisation

Grafikart 31 vidéos

Découvrir le CSS

Le CSS (Cascading Style Sheets) est le langage utilisé pour styliser et mettre en page les pages web. Cette formation extensive vous apprend à contrôler les couleurs, polices, espacements, positionnements, et à créer des designs responsives qui fonctionnent sur tous les appareils. Vous maîtriserez le modèle de boîte, flexbox, les layouts en grille, les animations et les techniques CSS modernes. Apprenez à transformer du HTML brut en sites web visuellement époustouflants et professionnels avec une hiérarchie de styles appropriée, des media queries pour la responsivité mobile, et les meilleures pratiques CSS qui rendent votre code maintenable et évolutif.

JS : Entre l'humain et la machine

Grafikart 47 vidéos

Apprendre JavaScript

JavaScript est le langage de programmation qui donne vie aux pages web avec interactivité et comportement dynamique. Cette formation complète couvre les variables, types de données, fonctions, boucles, conditions, manipulation du DOM, gestion des événements et les fonctionnalités modernes ES6+. Vous apprendrez à valider des formulaires, créer des interfaces utilisateur interactives, récupérer des données depuis des API, gérer les opérations asynchrones avec les promesses et async/await, et comprendre les concepts fondamentaux de la programmation. JavaScript alimente tout, des simples animations aux applications web complexes, en faisant une compétence essentielle pour tout développeur web.

Angular : Du web au mobile

Grafikart 2h 52min

Angular - Tutoriel complet

Angular est un framework puissant basé sur TypeScript, développé par Google pour construire des applications single-page évolutives (SPA). Cette formation couvre les composants, modules, services, injection de dépendances, routage, formulaires, communication HTTP et gestion d'état. Vous apprendrez l'architecture d'Angular, comprendrez les bases de TypeScript, travaillerez avec RxJS pour la programmation réactive, et construirez des applications prêtes pour la production avec une structure appropriée et les meilleures pratiques. Les outils complets d'Angular et son typage fort le rendent idéal pour les applications d'entreprise, fournissant tout le nécessaire pour développer, tester et déployer des applications web et mobiles modernes efficacement.
